The Task

We were tasked with creating a full stack web application for Panda Express that can handle any single restaurants orders, functions, reporting, and more.

Why?

Every restaurant needs a POS system, a way to manage inventory, and a kiosk/kitchen to streamline orders. However, oftentimes this is done through several different applications. It would be incredibly beneficial to have all the functions in one place.

Our Project

Utilized the T3 Tech Stack environment

  • An environment that uses Next.js, Typescript, tRPC, Prisma (Postgres DB connection), Tailwind, NextAuth
  • Used the Agile method of development
    • Project separated into 3 sprints where we did all of our coding to provide a Minimum Viable Product (MVP)
    • Had 1 week preparation periods in between to reflect, refactor, and plan the next sprint
  • Used Custom Google Sheet for Project Management
    • While it is not as flashy as Jira or Azure, it was a simple, free, and highly customizable way for us to see where we were at Most of us were very familiar with G-Suite which allowed us to utilize this sheet easily

Why us?

  • Many of us have worked in restaurants before and designed this project with that in mind
  • Sequential and simple logic
  • Pages are not cluttered
  • Not flashy - enables the worker to do their best work
  • Has added features that provides great utility for the user
  • Efficient use of database management
